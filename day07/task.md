k8sタスク（5時間）
scheduler-plugins のout-of-treeプラグイン作成方法に従って、超簡単なFilterプラグインを自作する。題材：「Podのアノテーション my-scheduler/require-label: <key>=<value> があったら、そのラベルを持たないnodeをFilterで弾く」。nodeAffinityでできることを敢えてプラグインとして書く、という練習。
ビルドして、kindにデプロイして、実際にPodが期待通りにフィルタされることを確認。

## Deploymentについて
- 自作Filterのために次のケースを用意
```
┌────────┬──────────────────────────────────────────────────┬────────────────────────┬───────────────────────────────────────────────┐
│ ケース │               Pod のアノテーション               │     Node のラベル      │                   期待結果                    
├────────┼──────────────────────────────────────────────────┼────────────────────────┼───────────────────────────────────────────────┤     
│ 1      │ なし                                             │ -                      │ Success(通過)→ 普通にスケジュールされる       
├────────┼──────────────────────────────────────────────────┼────────────────────────┼───────────────────────────────────────────────┤
│ 2      │ my-scheduler/required-label: env=production あり │ env=production あり    │ Success → そのノードにスケジュールされる      │                      
├────────┼──────────────────────────────────────────────────┼────────────────────────┼───────────────────────────────────────────────┤
│ 3      │ my-scheduler/required-label: env=production あり │ ラベルなし or 値が違う │ Unschedulable → スケジュールされない(Pending) │
└────────┴──────────────────────────────────────────────────┴────────────────────────┴───────────────────────────────────────────────┘ 
```
